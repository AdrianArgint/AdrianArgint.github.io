My Projects

Programming Language: C

TetriBit and TetriPic are two projects that simulates the well-known game ”Tetris”. The first one uses a 64-bit number which represents a 8x8 map, set bits meaning filled cells, while the latter, uses BitMap Pictures for the map as well as the blocks.
BattleShip Game is also a replica for an actual online game. It uses ncurses library so that imitates a GUI in terminal. The player can choose his/her map while the computers’ map is randomly generated. The player has powers to help him win the battle (destroy in advance, destroy random ship etc).
ATM reproduces a withdrawal machine. It uses linked-list to store all the cards, with its pin, its amount and its history. In case of 3 in a row wrong pin, the card will get blocked and you have to go to the bank to restore it.
Linux-FileSystem impersonates the file system of a Linux operating system with its main command line commands such as ”ls”, ”cd [directory]”, ”cd ..”, ”pwd”, ”touch”, ”rm” and so on. Its implementation is based on trie for a better performance.
Router implements the functionality of a real-world router which receives ICMP and ARP packages and sends them to next hop or not, depending on the situation.

Programming Language: Java

Passengers Boarding, as a first project in Java, uses priority queue for embarking people depending on their needs/ticket/age. It uses priority class rewritten, as well as for node class.
SmartThermostat is used to optimize the heating of a home (can be apartment or house) by detailed analysis of temperature sensors. The project role was to get acquainted with java collection framework.
